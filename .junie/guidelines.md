## コードベースの前提

- Java 基礎 Vol.07「配列」の授業用教材です。
- `src/` の各 Java ファイルは、原則として独立した `main` メソッド付きプログラムです。
- `javac src/<ClassName>.java` と `java -cp src <ClassName>` で対象ファイルを個別に確認します。
- 一部ファイルは、教材として意図的にコンパイルエラーや実行時例外を含みます。
